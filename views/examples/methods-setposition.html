ChessboardJS examples Methods - setPosition(position[, useAnimation=true])

setPosition(position[, useAnimation=true])


DESCRIPTION

Sets the position on the boards. Leaves the board orientation intact.

JAVASCRIPT

board.setPosition('r1bqkbnr/pppp1ppp/2n5/1B2p3/4P3/5N2/PPPP1PPP/RNBQK2R');

IN ACTION

Sets the position to the defined string. The format parameter can be either a position string a fen string or a notation object.

setPosition(format) position(format, false)

Some examples, that you can use:

  • rnbqkbnr/pppppppp/8/8/8/8/PPPPPPPP/RNBQKBNR
  • rnbqkbnrpppppppp00000000000000000000000000000000PPPPPPPPRNBQKBNR
  • r1bqkbnr/pppp1ppp/2n5/1B2p3/4P3/5N2/PPPP1PPP/RNBQK2R
  • 8/8/8/8/8/8/8/8

NOTES

Alternatively you can call the position() method as is in the example above.
