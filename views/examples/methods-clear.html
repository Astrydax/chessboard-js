ChessboardJS examples Methods - clear()

clear()


DESCRIPTION

Clear clears the pieces from the table and sets the orientation of the table to ChessUtils.ORIENTATION.white.

JAVASCRIPT

board.clear();

IN ACTION

clear() Filling up the board again

NOTES

Alternatively you can call the setPosition() or position() methods with the ChessUtils.FEN.emptyId parameter. Note, that in this case the orientation is not changed.

position(ChessUtils.FEN.emptyId) setPosition(ChessUtils.FEN.emptyId)
